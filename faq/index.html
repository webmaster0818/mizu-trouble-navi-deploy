よくある質問（FAQ）

水道修理に関するよくある質問にお答えします。

よくある質問

Q.水道修理の料金相場はいくらですか？

A.水道修理の料金は症状や作業内容によって大きく異なります。蛇口のパッキン交換なら3,000~5,000円、トイレのつまり除去は5,000~40,000円、水漏れ修理は3,000~30,000円程度が相場です。必ず作業前に見積もりを取りましょう。

Q.深夜や休日でも対応してもらえますか？

A.クラシアン、水の救急隊、イースマイルなど大手業者は24時間365日対応しています。ただし、深夜・早朝（22時~8時）は割増料金（3,000~8,000円程度）がかかる場合があります。

Q.見積もり後にキャンセルできますか？

A.多くの業者は見積もり無料・キャンセル無料です。特にイースマイルはキャンセル料完全無料を明示しています。ただし、一部の業者では出張費が発生する場合もあるため、事前に確認しましょう。

Q.水道修理の業者はどう選べばいいですか？

A.1. 複数社から見積もりを取る 2. 料金体系が明確か確認 3. 24時間対応可能か 4. 施工保証があるか 5. 口コミ・評判を確認 6. 水道局指定工事店か確認 の6点がポイントです。

Q.水道局指定工事店とは何ですか？

A.各自治体の水道局から認定を受けた業者のことです。給水装置工事や排水設備工事を行うには、この指定が必要です。指定業者は一定の技術基準を満たしているため、安心して依頼できます。

Q.賃貸住宅の水トラブルは誰が費用を負担しますか？

A.一般的に、経年劣化による故障は大家（管理会社）負担、入居者の過失による故障は入居者負担です。まず管理会社に連絡して指示を仰ぎ、緊急時は止水栓を閉めて応急処置をしましょう。

Q.水漏れを放置するとどうなりますか？

A.水道代の増加（月1,000~5,000円以上）、床・壁の腐食、カビの発生、シロアリの被害、階下への漏水被害（マンションの場合）など、放置するほど被害が拡大します。早めの修理が重要です。

Q.自分で水道修理をしても大丈夫ですか？

A.パッキン交換やシャワーヘッド交換など簡単な作業はDIY可能です。ただし、配管の工事や給水装置の変更は水道法で資格が必要と定められており、素人の作業は漏水や水質汚染のリスクがあるため、専門業者に依頼しましょう。

Q.修理にかかる時間はどのくらいですか？

A.簡単な修理（パッキン交換等）は30分~1時間程度、トイレのつまり除去は1~2時間、配管の修理・交換は半日~1日が目安です。事前に業者に所要時間を確認しましょう。

Q.止水栓の場所が分かりません。

A.トイレは便器横の壁や床、キッチンはシンク下、洗面所は洗面台下にあることが多いです。見つからない場合は、水道メーター横の元栓（家全体の水を止める）を閉めましょう。

Q.クレジットカードで支払えますか？

A.クラシアン、水の救急隊、イースマイルはクレジットカード対応しています。水道屋本舗と生活水道センターも主要カードに対応。事前に利用可能なカードブランドを確認しておくと安心です。

Q.保険で水道修理費用をカバーできますか？

A.火災保険の「水漏れ」や「水濡れ」の特約で修理費用がカバーされる場合があります。特にマンションの場合、個人賠償責任保険で階下への被害を補償できることも。保険会社に確認してみましょう。

Q.見積もりと実際の費用が違うことはありますか？

A.作業中に想定外の問題が見つかった場合、追加費用が発生することがあります。信頼できる業者は、追加費用が必要になった時点で報告し、承諾を得てから作業を進めます。最初に「追加料金の有無」を確認しましょう。

Q.水道トラブルの予防方法はありますか？

A.1. 月1回のパイプクリーナー使用 2. 排水口ネットの設置 3. 油を直接排水に流さない 4. 冬場の凍結防止対策 5. 10年以上の蛇口は早めに交換 6. 定期的な排水管の高圧洗浄（2~3年に1回）が効果的です。

Q.悪質な水道修理業者を見分ける方法は？

A.1. 電話口で正確な料金を伝えず「見ないと分からない」を繰り返す 2. 公式サイトに会社情報がない 3. 水道局指定工事店でない 4. 契約を急かす 5. 見積書を出さない 6. 異常に安い価格で宣伝している、などが警戒サインです。
